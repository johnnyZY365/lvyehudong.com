# 旅业互动官网上传服务器说明

版本：0.1
日期：2026-07-08

## 上传方式

1. 将发布包 `lvyehudong.com-v0.1-server.zip` 上传到服务器。
2. 解压后，把压缩包里的全部文件放到网站根目录。
3. 确认网站根目录里可以直接看到：
   - `index.html`
   - `lvyehudong-home.css`
   - `script.js`
   - `version.json`
   - `assets` 文件夹
4. 浏览器访问域名首页，确认页面正常显示。

## 注意

- 不要只上传 `index.html`，图片和样式都在同一个包里。
- 如果服务器已有旧文件，建议先备份旧网站目录，再覆盖上传。
- 上传后如果页面还是旧版，清理浏览器缓存或服务器/CDN 缓存。
